# ADNIMERGE2 <a href="https://adni.loni.usc.edu/"><img src="man/figures/logo.png" align="right" height="138" /></a>

This is an R data package for the Alzheimer’s Disease Neuroimaging
Initiative (ADNI) study. All datasets are included as shared on
<https://adni.loni.usc.edu/>.

## Package Access

The ADNI study R data package, `ADNIMERGE2`, is available on the data
sharing platform at [LONI website](https://adni.loni.usc.edu/). `ADNIMERGE2` R package is located in *Study Files \> Data & Database*
folder at data-sharing platform. To request access of the R package, it
is required to submit an online application via LONI website and
acceptance of the ADNI Data Use Agreement
[here](https://adni.loni.usc.edu/wp-content/uploads/how_to_apply/ADNI_Data_Use_Agreement.pdf).

## Installation

To install the package locally, run
`install.packages("path/to/ADNIMERGE2.tar.gz", repos = NULL, type = "source")`.

## Package Version

To see the actual package version, run `packageVersion("ADNIMERGE2")`
after installing the R package locally.
